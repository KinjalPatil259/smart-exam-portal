About Smart Online Exam System

A unified online examination framework designed to eliminate the logistical constraints of physical test hosting while maintaining rigorous standards of assessment security and administrative oversight.

System Features

  • ✓ Custom Session-based Security
  • ✓ Glassmorphic Responsive Interface
  • ✓ Random Question Allocation
  • ✓ Multi-role administrative dashboard
  • ✓ Subject-wise metrics rendering
  • ✓ Negative Marking calculations
  • ✓ Direct PDF Scorecard downloading
  • ✓ Microsoft Excel export capability

Development Stack

Built with a clean Python Flask MVC layout. Database queries are handled dynamically using SQLAlchemy. The frontend utilizes responsive vanilla HTML5, CSS3, and JavaScript, backed by Chart.js for reports.
